# 📝 Notes — Personal Reflections

Yahan tum apne personal "aha moments", interview experiences, doubts, aur reflections likho.

## Convention

**File naming**: `day-XXX.md` (matches lesson day number)

**Suggested structure per note**:
```markdown
# Day XXX — [Topic]

## ✨ Aha Moments
- [What clicked today that didn't before]

## 🤔 Confusions
- [Things still unclear, need to revisit]

## 🎯 Real Project Application
- [How can I apply this in my current Java/Spring/Angular work?]

## 💡 Connections
- [How does this relate to previous days?]

## 🗣️ Practice Recording
- Did I record myself answering the interview question today? Y/N
- Confidence rating: ⭐⭐⭐☆☆
```

## When To Write Notes

- ✅ After completing the lesson (5-10 min reflection)
- ✅ When something clicks unexpectedly
- ✅ When you face a confusion you need to revisit
- ✅ After a real interview where today's concept came up

## Why Separate From Lessons?

Lessons = curriculum content (mentor-generated, comprehensive).
Notes = your brain's response to it (personal, raw, growth tracker).

Future-you will thank present-you when reviewing for FAANG interviews.
