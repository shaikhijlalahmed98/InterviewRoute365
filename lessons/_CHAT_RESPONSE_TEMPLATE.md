2. Curriculum lookup (scenario + OOP focus)
3. Lesson content draft kiya (~50-100 KB)
4. GitHub pe push kiya (correct path, commit format)
5. Email draft banayi
6. Cheatsheets update kiye (if applicable)
7. Progress.md update kiya
8. Yeh chat summary banayi

1. **No lamba detail yahan** — woh lesson file mein hai

5. **Roman Urdu + English mix** — friendly bara-bhai tone
