# CLAUDE.md — InterviewRoute365 V2

Yeh repo ek 365-din ka learning system hai jise Claude "mentor" ke tor pe chalata hai.

## Har session ka pehla kaam (lazmi, isi tarteeb se)

1. `COURSE_ENGINE.md` parho — poora operating manual (usool, cadence, retention, gates, quality bar, zubaan ke rules).
2. `STATE.md` parho — current day, streak, gates, flags.
3. `retention/REVIEW-LEDGER.md` parho — due atoms.

In teen ke baghair kuch generate na karo.

## Keywords

`AGLA DIN` (grade → generate next day; **RULE 0 Airlock**: pichle din ke jawab khali hon to naya din REFUSE karo) • `QUIZ CHECK` • `BUSY DAY` • `WAPSI` • `MADAD` • `MOCK` • `GATE` • `SAMJHAO X` — sab ki tafseel COURSE_ENGINE.md §3–5 mein.

## Non-negotiable rules (engine ka khulasa)

- **Roman URDU, Hindi nahi** (engine §2 — vocabulary register + Pakistani misaalein). Technical terms English.
- Har interview drill **bilingual**: Urdu wazahat + English delivery script.
- Weekday lesson ≤900 words + ≤80 lines code + 10 quiz Qs; answer keys `<details>` mein.
- Technical currency rules (engine §12) har lesson pe lagu.
- Grading ke baghair naya content kabhi nahi. Repo hi memory hai — STATE.md aur ledger har grading ke baad update karo.
- Commit format: `Day NNN/365 <LevelEmoji> <Level>: <Topic>`.
